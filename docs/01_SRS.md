# Software Requirements Specification (SRS)

## Project Name

LCG Commerce

## Business Name

Lone Carpet Gallery

## Project Goal

Develop a modern e commerce platform for Lone Carpet Gallery that enables customers across Kashmir to browse products, place orders, and contact the store. The platform will also provide an admin dashboard to manage products, categories, inventory, and customer orders.

## Users

### Customer

Browse products

Search products

Filter by category

View product details

Add products to cart

Place orders

Track order status

Contact the shop

### Admin

Secure login

Manage products

Manage categories

Manage inventory

Manage orders

Manage customers

View dashboard statistics
